# Dataset source

`pizza_place_sales_archive.zip` contains the four CSV files supplied for this challenge: `orders.csv`, `order_details.csv`, `pizzas.csv`, and `pizza_types.csv`.

Source: [Pizza Place Sales on Kaggle](https://www.kaggle.com/datasets/mysarahmadbhat/pizza-place-sales). The archive is committed so a reviewer can run the import without a Kaggle account or an additional download.
